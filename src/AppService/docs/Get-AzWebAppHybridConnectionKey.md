# Get-AzWebAppHybridConnectionKey

## SYNOPSIS
Gets the send key name and value for a Hybrid Connection.

## SYNTAX

```
Get-AzWebAppHybridConnectionKey -Name <String> -NamespaceName <String> -RelayName <String>
 -ResourceGroupName <String> -SubscriptionId <String[]> [-DefaultProfile <PSObject>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Gets the send key name and value for a Hybrid Connection.
