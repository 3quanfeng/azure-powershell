# Test-AzWebAppCloneable

## SYNOPSIS
Shows whether an app can be cloned to another resource group or subscription.

## SYNTAX

```
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

```
 [<CommonParameters>]
```
