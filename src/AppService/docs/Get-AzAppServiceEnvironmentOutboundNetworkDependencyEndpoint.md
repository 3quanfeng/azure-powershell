# Get-AzAppServiceEnvironmentOutboundNetworkDependencyEndpoint

## SYNOPSIS
Get the network endpoints of all outbound dependencies of an App Service Environment.

## SYNTAX

### Get (Default)
```
Get-AzAppServiceEnvironmentOutboundNetworkDependencyEndpoint -Name <String> -ResourceGroupName <String>
 -SubscriptionId <String[]> [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### GetViaIdentity
```
Get-AzAppServiceEnvironmentOutboundNetworkDependencyEndpoint -InputObject <IWebSiteIdentity>
 [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## DESCRIPTION
Get the network endpoints of all outbound dependencies of an App Service Environment.
