# Remove-AzWebAppPremierAddOn

## SYNOPSIS
Delete a premier add-on from an app.

## SYNTAX

### Delete (Default)
```
Remove-AzWebAppPremierAddOn -Name <String> -PremierAddOnName <String> -ResourceGroupName <String>
 -SubscriptionId <String> [-PassThru] [-DefaultProfile <PSObject>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzWebAppPremierAddOn -InputObject <IWebSiteIdentity> [-PassThru] [-DefaultProfile <PSObject>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Delete a premier add-on from an app.
