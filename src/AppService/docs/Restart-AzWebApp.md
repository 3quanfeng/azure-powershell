# Restart-AzWebApp

## SYNOPSIS
Restarts an app (or deployment slot, if specified).

## SYNTAX

### Restart (Default)
```
Restart-AzWebApp -Name <String> -ResourceGroupName <String> -SubscriptionId <String> [-SoftRestart <Boolean>]
 [-Synchronou <Boolean>] [-PassThru] [-DefaultProfile <PSObject>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RestartViaIdentity
```
Restart-AzWebApp -InputObject <IWebSiteIdentity> [-SoftRestart <Boolean>] [-Synchronou <Boolean>] [-PassThru]
 [-DefaultProfile <PSObject>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Restarts an app (or deployment slot, if specified).
