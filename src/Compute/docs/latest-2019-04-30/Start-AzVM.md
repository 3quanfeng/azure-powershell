# Start-AzVM

## SYNOPSIS
The operation to start a virtual machine.

## SYNTAX

### Start1 (Default)
```
Start-AzVM -Name <String> -ResourceGroupName <String> -SubscriptionId <String> [-PassThru]
 [-DefaultProfile <PSObject>] [-AsJob] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### StartViaIdentity1
```
Start-AzVM -InputObject <IComputeIdentity> [-PassThru] [-DefaultProfile <PSObject>] [-AsJob] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
The operation to start a virtual machine.
