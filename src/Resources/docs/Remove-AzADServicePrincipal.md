# Remove-AzADServicePrincipal

## SYNOPSIS

## SYNTAX

```
Remove-AzADServicePrincipal -ObjectId <String> -TenantId <String> [-PassThru] [-DefaultProfile <PSObject>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Deletes a service principal from the directory.
