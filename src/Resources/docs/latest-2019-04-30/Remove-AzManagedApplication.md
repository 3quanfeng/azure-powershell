# Remove-AzManagedApplication

## SYNOPSIS
Deletes the managed application.

## SYNTAX

### Delete1 (Default)
```
Remove-AzManagedApplication -Id <String> [-PassThru] [-DefaultProfile <PSObject>] [-AsJob] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```

### Delete
```
Remove-AzManagedApplication -Name <String> -ResourceGroupName <String> -SubscriptionId <String> [-PassThru]
 [-DefaultProfile <PSObject>] [-AsJob] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity1
```
Remove-AzManagedApplication -InputObject <IResourcesIdentity> [-PassThru] [-DefaultProfile <PSObject>]
 [-AsJob] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzManagedApplication -InputObject <IResourcesIdentity> [-PassThru] [-DefaultProfile <PSObject>]
 [-AsJob] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Deletes the managed application.
