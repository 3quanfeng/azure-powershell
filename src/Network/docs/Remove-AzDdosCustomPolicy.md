# Remove-AzDdosCustomPolicy

## SYNOPSIS
Deletes the specified DDoS custom policy.

## SYNTAX

### Delete (Default)
```
Remove-AzDdosCustomPolicy -Name <String> -ResourceGroupName <String> -SubscriptionId <String> [-PassThru]
 [-DefaultProfile <PSObject>] [-AsJob] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzDdosCustomPolicy -InputObject <INetworkIdentity> [-PassThru] [-DefaultProfile <PSObject>] [-AsJob]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Deletes the specified DDoS custom policy.
