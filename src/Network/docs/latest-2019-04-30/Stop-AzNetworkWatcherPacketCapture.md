# Stop-AzNetworkWatcherPacketCapture

## SYNOPSIS
Stops a specified packet capture session.

## SYNTAX

### Stop (Default)
```
Stop-AzNetworkWatcherPacketCapture -Name <String> -NetworkWatcherName <String> -ResourceGroupName <String>
 -SubscriptionId <String> [-PassThru] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### StopViaIdentity
```
Stop-AzNetworkWatcherPacketCapture -InputObject <INetworkIdentity> [-PassThru] [-DefaultProfile <PSObject>]
 [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Stops a specified packet capture session.
