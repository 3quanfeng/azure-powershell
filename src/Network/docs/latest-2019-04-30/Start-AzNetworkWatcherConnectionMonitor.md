# Start-AzNetworkWatcherConnectionMonitor

## SYNOPSIS
Starts the specified connection monitor.

## SYNTAX

### Start (Default)
```
Start-AzNetworkWatcherConnectionMonitor -Name <String> -NetworkWatcherName <String>
 -ResourceGroupName <String> -SubscriptionId <String> [-PassThru] [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### StartViaIdentity
```
Start-AzNetworkWatcherConnectionMonitor -InputObject <INetworkIdentity> [-PassThru]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Starts the specified connection monitor.
