# Backup-AzKeyVaultCertificate

## SYNOPSIS
Requests that a backup of the specified certificate be downloaded to the client.
All versions of the certificate will be downloaded.
This operation requires the certificates/backup permission.

## SYNTAX

### Backup (Default)
```
Backup-AzKeyVaultCertificate -Name <String> -OutFile <String> [-KeyVaultDnsSuffix <String>]
 [-VaultName <String>] [-PassThru] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### BackupViaIdentity
```
Backup-AzKeyVaultCertificate -InputObject <IKeyVaultIdentity> -OutFile <String> [-KeyVaultDnsSuffix <String>]
 [-VaultName <String>] [-PassThru] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Requests that a backup of the specified certificate be downloaded to the client.
All versions of the certificate will be downloaded.
This operation requires the certificates/backup permission.
